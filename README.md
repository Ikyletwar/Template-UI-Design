## 🐛 Troubleshooting

### ❌ Animasi Tidak Jalan

**Cek:**
1. Browser support (Chrome/Firefox/Edge terbaru)
2. JavaScript enabled
3. Console errors (F12 → Console)

**Fix:**
```javascript
// Cek browser support
if (!window.requestAnimationFrame) {
  alert('Browser tidak support animation');
}
```

### ❌ Particles Tidak Muncul

**Cek:**
1. Canvas element ada (line ~410)
2. Function `initParticles()` dipanggil
3. Tidak ada JS errors

**Fix:**
```javascript
// Tambah di akhir script
console.log('Particles:', particles.length);
```

### ❌ Warna Tidak Berubah

**Cek:**
1. Edit di `:root` CSS variable
2. Cache browser (Ctrl+F5 / Cmd+Shift+R)
3. Tidak ada typo di variable name

**Fix:**
```css
/* Pastikan format benar */
--accent: #3b82f6;  /* Bukan "#3b82f" atau "3b82f6" */
```

### ❌ Deploy Tidak Work

**Cek:**
1. File `index.html` ada di root repository
2. GitHub Pages enabled
3. Branch benar (main/master)

**Fix:**
```bash
# Cek struktur
git ls-files
# Harus ada: index.html
```

### ❌ Mobile Tampilan Rusak

**Cek:**
1. Viewport meta tag ada (line 5)
2. Tailwind CDN loaded
3. Test di Chrome DevTools mobile view

**Fix:**
```html
<!-- Pastikan ada di <head> -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## ⚡ Performance Tips

### 1. Optimize untuk Low-End Devices

```javascript
// Detect device
const isLowEnd = navigator.hardwareConcurrency <= 4;

if (isLowEnd) {
  // Kurangi particles
  const particleCount = 30;
  // Matikan complex animations
  document.body.classList.add('reduced-motion');
}
```

### 2. Lazy Load Images (Jika Ada)

```html
<img src="hero.jpg" loading="lazy" alt="Hero">
```

### 3. Preload Critical Assets

```html
<link rel="preload" href="https://fonts.googleapis.com/css2?family=DM+Sans" as="style">
```

### 4. Minify HTML (Optional)

```bash
# Gunakan HTMLMinifier
npm install -g html-minifier
html-minifier --collapse-whitespace --minify-css --minify-js index.html -o index.min.html
```

## ❓ FAQ Super Lengkap

### Q: Apakah gratis untuk digunakan?
**A:** Ya! 100% gratis untuk project personal atau komersial. License MIT.

### Q: Apakah boleh dijual lagi?
**A:** Boleh! Template ini bisa digunakan untuk client atau dijual sebagai bagian dari produk kamu. Tapi tidak boleh dijual sebagai "template" saja (harus ada value tambahannya seperti customization, integration, dll).

### Q: Bisa ditambah backend?
