api.js
import { Zero } from '@zero/ai';
const client = new Zero({ apiKey: process.env.ZERO_KEY });
// Hasilkan respons cerdas dalam satu baris
const response = await client.generate({
prompt: "Jelaskan komputasi kuantum",
model: "zero-pro"
});
console.log(response.text);

Segala yang Anda butuhkan untuk meluncurkan AI

Dari prototipe hingga produksi, Zero menyediakan fondasi untuk aplikasi cerdas.

Sangat Cepat

Inferensi di edge dengan latensi di bawah 50ms secara global. Pengguna Anda tidak perlu menunggu kecerdasan.

Pengembang Utama

API bersih, SDK komprehensif, dan dokumentasi lengkap. Bangun dengan alat yang Anda cintai.

Siap Perusahaan

Sesuai SOC2, siap GDPR, dengan kontrol akses lanjutan dan log audit.

Otomatis Skalakan

Tangani lonjakan traffic tanpa hambatan. Dari nol hingga jutaan permintaan dengan mulus.

Perpustakaan Model

Akses 50+ model terlatih atau sesuaikan model Anda sendiri. Satu API, kemungkinan tak terbatas.

Analitik Real-time

Pantau penggunaan, biaya, dan performa secara real-time. Optimalkan dengan wawasan yang dapat ditindaklanjuti.
